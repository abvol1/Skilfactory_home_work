Слово "Итого" ищем в столбце B (индекс 1).

Суммируем столбцы: 4 (индекс 3) и 5 (индекс 4). Но в текущем коде для РВП указаны столбцы {2,5,6,7,9,10}. Если нужно только 4 и 5, заменим массив.

Лист 2 (ПОТ):

Слово "Итого" ищем в столбце A (индекс 0).

Суммируем столбцы: 4 (индекс 3) и 5 (индекс 4). В текущем коде для ПОТ указаны {9,10}. Также заменим.

Внесём исправления в метод insertTotalFormulas:

java
private void insertTotalFormulas(Worksheet sheet, List<RowData> rows, int startRowIndex, boolean isRvp) {
    if (rows.isEmpty()) return;

    int totalSearchColumn;
    int[] columnsToSum;

    if (isRvp) {
        totalSearchColumn = 1; // столбец B (индекс 1)
        // Суммируем столбцы 4 (индекс 3) и 5 (индекс 4) - как просили
        columnsToSum = new int[]{3, 4};
    } else {
        totalSearchColumn = 0; // столбец A (индекс 0)
        // Суммируем столбцы 4 (индекс 3) и 5 (индекс 4)
        columnsToSum = new int[]{3, 4};
    }

    // Ищем строку "Итого" начиная с конца данных
    int totalRow = findTotalRow(sheet, startRowIndex + rows.size(), totalSearchColumn);
    if (totalRow == -1) return;

    int firstDataRow = startRowIndex;
    int lastDataRow = startRowIndex + rows.size() - 1;

    for (int col : columnsToSum) {
        Cell totalCell = sheet.getCells().get(totalRow, col);
        String colLetter = CellsHelper.cellIndexToName(col, 0);
        String formula = String.format("=SUM(%s%d:%s%d)",
                colLetter, firstDataRow + 1,
                colLetter, lastDataRow + 1);
        totalCell.setFormula(formula);
    }
}

Для РВП: RVP_START_ROW = 17 (18-я строка).

Для ПОТ: POT_START_ROW = 7 (8-я строка).

Метод findTotalRow корректно ищет подстроку "итого" в указанном столбце, начиная с первой строки после данных. Если слово "Итого" находится сразу под таблицей (без пустых строк), формула вставится правильно.

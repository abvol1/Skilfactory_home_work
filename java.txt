
Генератор распоряжений из файла CZO
Мы получили несколько изображений с текстом технического задания на Java-приложение с JavaFX и Scene Builder для генерации распоряжений на основе Excel-файла CZO_RES_BASE_DZ_New_DBI.xlsx. Задача: создать простой код с максимумом комментариев, обработка файла CZO, группировка по филиалу в столбце B (первые 3 цифры значений 001-003, 005-009), формирование отдельных распоряжений для каждой группы.

Основное содержание ТЗ:

Отчет CZO_RES_BASE_DZ_New_DBI содержит столбцы (из изображений и описания). Нужно создать две вкладки в результирующем Excel-файле распоряжения:

"Распоряжение (РВП)"

"Расчетная база ПОТ"

Критерии отбора строк из исходного файла для каждой вкладки:

"Распоряжение (РВП)": счета 60323, 458, 47423*, у которых столбец "Расхождение" (16) не равен 0.

"Расчетная база ПОТ":

для счетов 60323*: если столбец "Остаток счета РБ на дату резервирования" (7) > 0.

для счетов 458, 47423: если столбец (7) или (8) "Остаток счета просроченной РБ на дату резервирования" > 0.

Столбцы исходного файла (по нумерации из описания):

(1) № п/п

(2) Количество

(3) Состояние

(4) Счёт

(5) Валюта

(6) Курс валюты

(7) Остаток счета РБ на дату резервирования

(8) Остаток счета просроченной РБ на дату резервирования (в одном месте написано "Остаток счета РБ на дату резервирования" дважды, но судя по описанию второй - это столбец 8)

(9) Остаток прочих номиналов РБ на дату резервирования

(10) Сумма оборотов К-т

(11) Сумма оборотов Д-т

(12) Счет резерва

(13) Примечания

(14) Тип резерва

(15) Остаток счета резерва на дату резервирования

(16) Расчетный резерв (или "Расчетный размер"?)

(17) Расхождение

(18) Дата отчисления

В другом месте указано "Расчетный резерв" (15) и "Расхождение" (16) и т.д. Надо уточнить индексы. Но для реализации будем ориентироваться на описание: есть столбцы "Остаток счета РБ на дату резервирования" (7), "Остаток счета просроченной РБ на дату резервирования" (8), "Счет РБ" (5), "Остаток счета резерва на дату резервирования" (14), "Расчетный резерв" (15), "Расхождение" (16).

Дополнительно есть колонка B с филиалом в исходном файле CZO, значения типа "001-003", "005-009" и т.д. Нужно группировать по первым трем цифрам. Создавать отдельные файлы распоряжений для каждой группы.

Для вкладки "Расчетная база ПОТ" надо заполнить таблицу с колонками:

ФИО клиента (из столбца "Клиент" (3) исходного отчета)

Номер договора (? не указано в исходном, возможно из другого места)

Дата заключения договора

Признак принадлежности к ПОТ на дату ...

Валюта (из столбца "Валюта счета" (6))

Коэффициент резервирования (рассчитывается по правилам)

Категория качества (рассчитывается)

Счет для отчетности (из "Счет РБ" (5))

Остаток по счету ЭРБ (из разных столбцов в зависимости от счета)

Остаток по счету ЭРБ до 30 дней (из (7))

Длительность задолженности (?)

Но в задании просят "сосредоточиться не над оформлением распоряжений а над содержанием таблиц, первая строка у таблиц начинается с 9ой строки". Также "в коде прошу тебя сосредоточится не над оформлением распоряжений а над содержанием таблиц".

То есть нам не нужно оформлять шапки и тексты, только заполнять данные в строках, начиная с 9 строки.

Мы должны предоставить простой JavaFX код с использованием Scene Builder, с диалогом выбора файла CZO, обработкой Excel через Apache POI.

Учитывая объем ТЗ, мы предложим упрощенную структуру, фокусируясь на чтении исходного файла, фильтрации, группировке по филиалу, создании новых книг Excel с двумя листами и заполнении таблиц данными по описанным алгоритмам, с комментариями.

Ответ должен содержать код на Java с пояснениями.

Ниже приведён простой, хорошо закомментированный код JavaFX-приложения с использованием Apache POI для работы с Excel. Приложение позволяет выбрать файл CZO_RES_BASE_DZ_New_DBI.xlsx, группирует записи по первым трём цифрам столбца B («Филиал»), фильтрует данные согласно ТЗ и формирует отдельные файлы распоряжений для каждой группы. В каждом файле создаются две вкладки: «Распоряжение (РВП)» и «Расчетная база ПОТ», таблицы заполняются начиная с 9-й строки (первые 8 строк оставлены пустыми — в них можно позже вставить шапку и текст).

// ======================= Controller.java =======================
import javafx.fxml.FXML;
import javafx.scene.control.Alert;
import javafx.scene.control.Button;
import javafx.stage.FileChooser;
import java.io.File;

public class Controller {

    @FXML
    private Button selectFileButton;

    @FXML
    private void initialize() {
        selectFileButton.setOnAction(event -> chooseFileAndProcess());
    }

    private void chooseFileAndProcess() {
        FileChooser fileChooser = new FileChooser();
        fileChooser.setTitle("Выберите файл CZO");
        fileChooser.getExtensionFilters().add(
                new FileChooser.ExtensionFilter("Excel files", "*.xlsx", "*.xls")
        );
        File selectedFile = fileChooser.showOpenDialog(selectFileButton.getScene().getWindow());

        if (selectedFile != null) {
            try {
                CzoProcessor processor = new CzoProcessor();
                processor.process(selectedFile);
                showInfo("Обработка завершена", "Файлы распоряжений созданы в папке с исходным файлом.");
            } catch (Exception e) {
                showError("Ошибка обработки", e.getMessage());
                e.printStackTrace();
            }
        }
    }

    private void showInfo(String title, String message) {
        Alert alert = new Alert(Alert.AlertType.INFORMATION);
        alert.setTitle(title);
        alert.setHeaderText(null);
        alert.setContentText(message);
        alert.showAndWait();
    }

    private void showError(String title, String message) {
        Alert alert = new Alert(Alert.AlertType.ERROR);
        alert.setTitle(title);
        alert.setHeaderText(null);
        alert.setContentText(message);
        alert.showAndWait();
    }
}

// ======================= CzoProcessor.java =======================
import org.apache.poi.ss.usermodel.*;
import org.apache.poi.xssf.usermodel.XSSFWorkbook;

import java.io.File;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.io.IOException;
import java.math.BigDecimal;
import java.util.*;
import java.util.regex.Pattern;

public class CzoProcessor {

    // Индексы столбцов в исходном файле (начиная с 0)
    private static final int COL_FILIAL = 1;            // B - филиал (первые 3 цифры)
    private static final int COL_CLIENT_NAME = 2;       // C - ФИО клиента (столбец "Клиент")
    private static final int COL_ACCOUNT_RB = 4;        // E - Счет РБ (номер лицевого счёта)
    private static final int COL_CURRENCY = 5;          // F - Валюта счета
    private static final int COL_BALANCE_RB = 6;        // G - Остаток счета РБ на дату резервирования (7)
    private static final int COL_OVERDUE_BALANCE = 7;   // H - Остаток счета просроченной РБ (8)
    // COL_RESERVE_ACCOUNT = ? (столбец 12 - Счет резерва, индекс 11)
    private static final int COL_RESERVE_BALANCE = 14;  // O - Остаток счета резерва на дату резервирования (15)
    private static final int COL_CALC_RESERVE = 15;     // P - Расчетный резерв (16)
    private static final int COL_DISCREPANCY = 16;      // Q - Расхождение (17)

    // Шаблоны счетов
    private static final Pattern ACCOUNT_60323 = Pattern.compile("^60323.*");
    private static final Pattern ACCOUNT_458 = Pattern.compile("^458.*");
    private static final Pattern ACCOUNT_47423 = Pattern.compile("^47423.*");

    /**
     * Основной метод обработки.
     */
    public void process(File inputFile) throws IOException {
        // 1. Читаем исходный файл
        Map<String, List<RowData>> groupedByFilial = readAndGroupByFilial(inputFile);

        // 2. Для каждой группы создаём отдельный файл распоряжения
        for (Map.Entry<String, List<RowData>> entry : groupedByFilial.entrySet()) {
            String filialCode = entry.getKey();
            List<RowData> rows = entry.getValue();
            createOrderFile(inputFile.getParentFile(), filialCode, rows);
        }
    }

    /**
     * Чтение исходного Excel и группировка по филиалу (первые 3 цифры столбца B).
     */
    private Map<String, List<RowData>> readAndGroupByFilial(File file) throws IOException {
        Map<String, List<RowData>> groups = new HashMap<>();

        try (FileInputStream fis = new FileInputStream(file);
             Workbook workbook = WorkbookFactory.create(fis)) {

            Sheet sheet = workbook.getSheetAt(0); // предполагаем, что данные на первом листе
            DataFormatter formatter = new DataFormatter();

            // Пропускаем заголовок (предположим, что первая строка - шапка)
            for (int rowIdx = 1; rowIdx <= sheet.getLastRowNum(); rowIdx++) {
                Row row = sheet.getRow(rowIdx);
                if (row == null) continue;

                // Извлекаем значение филиала (столбец B)
                Cell filialCell = row.getCell(COL_FILIAL);
                String filialRaw = formatter.formatCellValue(filialCell).trim();
                if (filialRaw.isEmpty()) continue;

                // Берём первые 3 символа (например, из "001-003" -> "001")
                String filialKey = filialRaw.length() >= 3 ? filialRaw.substring(0, 3) : filialRaw;
                // Фильтруем только нужные филиалы (001-003, 005-009)
                if (!isTargetFilial(filialKey)) continue;

                // Читаем остальные поля
                RowData data = new RowData();
                data.accountRB = formatter.formatCellValue(row.getCell(COL_ACCOUNT_RB));
                data.clientName = formatter.formatCellValue(row.getCell(COL_CLIENT_NAME));
                data.currency = formatter.formatCellValue(row.getCell(COL_CURRENCY));
                data.balanceRB = getNumericValue(row.getCell(COL_BALANCE_RB));
                data.overdueBalance = getNumericValue(row.getCell(COL_OVERDUE_BALANCE));
                data.reserveBalance = getNumericValue(row.getCell(COL_RESERVE_BALANCE));
                data.calcReserve = getNumericValue(row.getCell(COL_CALC_RESERVE));
                data.discrepancy = getNumericValue(row.getCell(COL_DISCREPANCY));

                // Добавляем в группу
                groups.computeIfAbsent(filialKey, k -> new ArrayList<>()).add(data);
            }
        }
        return groups;
    }

    /**
     * Проверка, входит ли код филиала в заданные диапазоны (001-003, 005-009).
     */
    private boolean isTargetFilial(String code) {
        try {
            int num = Integer.parseInt(code);
            return (num >= 1 && num <= 3) || (num >= 5 && num <= 9);
        } catch (NumberFormatException e) {
            return false;
        }
    }

    /**
     * Получение числового значения из ячейки (BigDecimal для точности).
     */
    private BigDecimal getNumericValue(Cell cell) {
        if (cell == null) return BigDecimal.ZERO;
        try {
            if (cell.getCellType() == CellType.NUMERIC) {
                return BigDecimal.valueOf(cell.getNumericCellValue());
            } else if (cell.getCellType() == CellType.STRING) {
                String str = cell.getStringCellValue().trim().replace(",", ".");
                return new BigDecimal(str.isEmpty() ? "0" : str);
            }
        } catch (Exception ignored) {}
        return BigDecimal.ZERO;
    }

    /**
     * Создание файла распоряжения для конкретного филиала.
     */
    private void createOrderFile(File parentDir, String filialCode, List<RowData> allRows) throws IOException {
        // Фильтрация строк для двух вкладок
        List<RowData> rvpRows = new ArrayList<>();
        List<RowData> potRows = new ArrayList<>();

        for (RowData row : allRows) {
            // Распоряжение (РВП): счета 60323*,458*,47423* и расхождение != 0
            if (isRvpAccount(row.accountRB) && row.discrepancy.compareTo(BigDecimal.ZERO) != 0) {
                rvpRows.add(row);
            }

            // Расчетная база ПОТ
            if (isPotAccount(row.accountRB)) {
                if (ACCOUNT_60323.matcher(row.accountRB).matches()) {
                    // для 60323* остаток по счёту РБ > 0
                    if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0) {
                        potRows.add(row);
                    }
                } else { // 458* или 47423*
                    // остаток РБ > 0 или остаток просроченной РБ > 0
                    if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0 ||
                        row.overdueBalance.compareTo(BigDecimal.ZERO) > 0) {
                        potRows.add(row);
                    }
                }
            }
        }

        // Создаём новую книгу Excel
        try (Workbook workbook = new XSSFWorkbook()) {
            // Создаём лист "Распоряжение (РВП)" и заполняем
            Sheet sheetRvp = workbook.createSheet("Распоряжение (РВП)");
            fillRvpSheet(sheetRvp, rvpRows);

            // Создаём лист "Расчетная база ПОТ"
            Sheet sheetPot = workbook.createSheet("Расчетная база ПОТ");
            fillPotSheet(sheetPot, potRows);

            // Сохраняем файл
            String outputFileName = String.format("Распоряжение_филиал_%s.xlsx", filialCode);
            File outputFile = new File(parentDir, outputFileName);
            try (FileOutputStream fos = new FileOutputStream(outputFile)) {
                workbook.write(fos);
            }
        }
    }

    private boolean isRvpAccount(String account) {
        return ACCOUNT_60323.matcher(account).matches() ||
               ACCOUNT_458.matcher(account).matches() ||
               ACCOUNT_47423.matcher(account).matches();
    }

    private boolean isPotAccount(String account) {
        return isRvpAccount(account); // те же счета
    }

        Workbook workbook = sheet.getWorkbook();
        CellStyle numberStyle = workbook.createCellStyle();
        numberStyle.setDataFormat(workbook.createDataFormat().getFormat("#,##0.00"));

            row.createCell(1).setCellValue(data.accountRB);

Уточняю так как в шаблонах под таблицами тоже есть нужный текст и его нельзя стирать то строки в таблицу должны предварительно вставлять
Мы должны уточнить реализацию: в шаблоне Excel уже есть две вкладки с шапками и текстом под таблицами. Нужно не просто перезаписывать строки с 9-й, а вставлять строки внутрь таблицы, не затрагивая нижний текст. В Excel нет понятия "таблица" как диапазона с автоматическим расширением, но можно вставить новые строки перед строками с текстом, чтобы сдвинуть текст вниз. Однако проще: предположим, что в шаблоне таблица занимает определённое количество строк (например, одна строка-образец для данных), а текст под таблицей находится на фиксированных строках (например, с 20-й и ниже). Мы можем определить, где заканчивается таблица и начинается текст (например, по ключевым словам или по номеру строки). Поскольку в ТЗ сказано "первая строка у таблиц начинается с 9-ой строки", это значит, что строки 1-8 заняты шапкой и заголовками, а с 9-й начинаются данные. Текст под таблицей будет ниже последней строки данных. Чтобы не затереть текст, нужно при заполнении сначала очистить все строки с данными (начиная с 9-й до строки перед началом текста), а затем вставить столько строк, сколько нужно для записей, копируя форматирование. Или можно создать новые строки между последней строкой данных шаблона и началом текста.

Упростим: в шаблоне заранее оставим только одну пустую строку для данных на 9-й строке, а весь текст ниже (например, с 10-й строки). Тогда алгоритм:

Открыть шаблон.

Найти индекс строки, с которой начинается текст под таблицей (можно определить по содержимому, например, искать строку с текстом "Итого" или "Примечание").

Удалить все строки между 9-й строкой (индекс 8) и строкой перед текстом.

Вставить нужное количество новых строк на место удалённых, скопировав стиль из оставшейся строки шаблона (9-й).

Заполнить вставленные строки данными.

Чтобы не усложнять, можно в коде зафиксировать, что текст под таблицей начинается, скажем, с 15-й строки (или определить по ключевому слову "итого"). Для простоты предположим, что на листе "Распоряжение (РВП)" таблица данных заканчивается перед строкой, содержащей слово "Итого:" (или другой маркер). А на листе "Расчетная база ПОТ" аналогично.

java
import org.apache.poi.ss.usermodel.*;
import org.apache.poi.ss.util.CellRangeAddress;
import org.apache.poi.xssf.usermodel.XSSFWorkbook;

import java.io.*;
import java.math.BigDecimal;
import java.math.RoundingMode;
import java.util.*;
import java.util.regex.Pattern;

public class CzoProcessor {

    // Индексы столбцов исходного файла (0-based)
    private static final int COL_FILIAL = 1;             // B - Филиал (первые 3 цифры)
    private static final int COL_CLIENT_NAME = 2;        // C - Клиент (ФИО)
    private static final int COL_ACCOUNT_RB = 4;         // E - Счет РБ (лицевой счет)
    private static final int COL_CURRENCY = 5;           // F - Валюта счета
    private static final int COL_BALANCE_RB = 6;         // G - Остаток счета РБ (7)
    private static final int COL_OVERDUE_BALANCE = 7;    // H - Остаток просроченной РБ (8)
    private static final int COL_RESERVE_BALANCE = 14;   // O - Остаток счета резерва (15)
    private static final int COL_CALC_RESERVE = 15;      // P - Расчетный резерв (16)
    private static final int COL_DISCREPANCY = 16;       // Q - Расхождение (17)

    // Шаблоны счетов
    private static final Pattern ACCOUNT_60323 = Pattern.compile("^60323.*");
    private static final Pattern ACCOUNT_458 = Pattern.compile("^458.*");
    private static final Pattern ACCOUNT_47423 = Pattern.compile("^47423.*");

    /**
     * Главный метод обработки.
     */
    public void process(File inputFile) throws IOException {
        // 1. Чтение и группировка исходных данных
        Map<String, List<RowData>> groupedByFilial = readAndGroupByFilial(inputFile);

        // 2. Для каждого филиала создаём файл на основе шаблона
        for (Map.Entry<String, List<RowData>> entry : groupedByFilial.entrySet()) {
            String filialCode = entry.getKey();
            List<RowData> rows = entry.getValue();
            createOrderFromTemplate(inputFile.getParentFile(), filialCode, rows);
        }
    }

    /**
     * Чтение исходного Excel, группировка по первым трём цифрам филиала.
     */
    private Map<String, List<RowData>> readAndGroupByFilial(File file) throws IOException {
        Map<String, List<RowData>> groups = new HashMap<>();

        try (FileInputStream fis = new FileInputStream(file);
             Workbook workbook = WorkbookFactory.create(fis)) {

            Sheet sheet = workbook.getSheetAt(0);
            DataFormatter formatter = new DataFormatter();

            for (int rowIdx = 1; rowIdx <= sheet.getLastRowNum(); rowIdx++) {
                Row row = sheet.getRow(rowIdx);
                if (row == null) continue;

                // Филиал (первые 3 символа столбца B)
                Cell filialCell = row.getCell(COL_FILIAL);
                String filialRaw = formatter.formatCellValue(filialCell).trim();
                if (filialRaw.isEmpty()) continue;

                String filialKey = filialRaw.length() >= 3 ? filialRaw.substring(0, 3) : filialRaw;
                if (!isTargetFilial(filialKey)) continue;

                RowData data = new RowData();
                data.accountRB = formatter.formatCellValue(row.getCell(COL_ACCOUNT_RB));
                data.clientName = formatter.formatCellValue(row.getCell(COL_CLIENT_NAME));
                data.currency = formatter.formatCellValue(row.getCell(COL_CURRENCY));
                data.balanceRB = getNumericValue(row.getCell(COL_BALANCE_RB));
                data.overdueBalance = getNumericValue(row.getCell(COL_OVERDUE_BALANCE));
                data.reserveBalance = getNumericValue(row.getCell(COL_RESERVE_BALANCE));
                data.calcReserve = getNumericValue(row.getCell(COL_CALC_RESERVE));
                data.discrepancy = getNumericValue(row.getCell(COL_DISCREPANCY));

                groups.computeIfAbsent(filialKey, k -> new ArrayList<>()).add(data);
            }
        }
        return groups;
    }

    private boolean isTargetFilial(String code) {
        try {
            int num = Integer.parseInt(code);
            return (num >= 1 && num <= 3) || (num >= 5 && num <= 9);
        } catch (NumberFormatException e) {
            return false;
        }
    }

    private BigDecimal getNumericValue(Cell cell) {
        if (cell == null) return BigDecimal.ZERO;
        try {
            if (cell.getCellType() == CellType.NUMERIC) {
                return BigDecimal.valueOf(cell.getNumericCellValue());
            } else if (cell.getCellType() == CellType.STRING) {
                String str = cell.getStringCellValue().trim().replace(",", ".");
                return new BigDecimal(str.isEmpty() ? "0" : str);
            }
        } catch (Exception ignored) {}
        return BigDecimal.ZERO;
    }

    /**
     * Создание файла распоряжения из шаблона template.xlsx.
     */
    private void createOrderFromTemplate(File parentDir, String filialCode, List<RowData> allRows) throws IOException {
        // Загружаем шаблон из ресурсов
        try (InputStream templateStream = getClass().getResourceAsStream("/template.xlsx")) {
            if (templateStream == null) {
                throw new IOException("Шаблон template.xlsx не найден в ресурсах!");
            }

            try (Workbook workbook = new XSSFWorkbook(templateStream)) {
                // Фильтрация строк для двух листов
                List<RowData> rvpRows = new ArrayList<>();
                List<RowData> potRows = new ArrayList<>();

                for (RowData row : allRows) {
                    // РВП: счета 60323*,458*,47423* и расхождение != 0
                    if (isRvpAccount(row.accountRB) && row.discrepancy.compareTo(BigDecimal.ZERO) != 0) {
                        rvpRows.add(row);
                    }

                    // ПОТ
                    if (isPotAccount(row.accountRB)) {
                        if (ACCOUNT_60323.matcher(row.accountRB).matches()) {
                            if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0) {
                                potRows.add(row);
                            }
                        } else { // 458* или 47423*
                            if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0 ||
                                row.overdueBalance.compareTo(BigDecimal.ZERO) > 0) {
                                potRows.add(row);
                            }
                        }
                    }
                }

                // Заполняем листы
                Sheet sheetRvp = workbook.getSheet("Распоряжение (РВП)");
                Sheet sheetPot = workbook.getSheet("Расчетная база ПОТ");
                if (sheetRvp == null || sheetPot == null) {
                    throw new IOException("В шаблоне отсутствуют нужные листы!");
                }

                fillSheetWithInsert(sheetRvp, rvpRows, true);
                fillSheetWithInsert(sheetPot, potRows, false);

                // Сохраняем результат
                String outputFileName = String.format("Распоряжение_филиал_%s.xlsx", filialCode);
                File outputFile = new File(parentDir, outputFileName);
                try (FileOutputStream fos = new FileOutputStream(outputFile)) {
                    workbook.write(fos);
                }
            }
        }
    }

    private boolean isRvpAccount(String account) {
        return ACCOUNT_60323.matcher(account).matches() ||
               ACCOUNT_458.matcher(account).matches() ||
               ACCOUNT_47423.matcher(account).matches();
    }

    private boolean isPotAccount(String account) {
        return isRvpAccount(account);
    }

    /**
     * Заполнение листа с динамической вставкой строк.
     * @param sheet лист
     * @param rows данные
     * @param isRvp true для листа "Распоряжение (РВП)", false для "Расчетная база ПОТ"
     */
    private void fillSheetWithInsert(Sheet sheet, List<RowData> rows, boolean isRvp) {
        // 1. Найти строку, с которой начинается текст под таблицей (после 8-й)
        int textStartRow = findTextStartRow(sheet);
        if (textStartRow == -1) {
            // Если не нашли, считаем, что текст начинается сразу после шаблонной строки (10)
            textStartRow = 9; // индекс 9 = 10-я строка
        }

        // 2. Удаляем все строки между 9-й (индекс 8) и textStartRow (не включая textStartRow)
        int firstDataRow = 8; // 9-я строка (индекс 8)
        if (textStartRow > firstDataRow) {
            for (int i = textStartRow - 1; i >= firstDataRow; i--) {
                Row row = sheet.getRow(i);
                if (row != null) {
                    sheet.removeRow(row);
                }
            }
        }

        // 3. Вставляем столько строк, сколько нужно для данных (если rows.size() > 1)
        int rowsNeeded = Math.max(rows.size(), 1);
        if (rowsNeeded > 1) {
            // Копируем стили из строки-образца (сейчас она на месте 9-й строки)
            Row templateRow = sheet.getRow(firstDataRow);
            if (templateRow == null) {
                templateRow = sheet.createRow(firstDataRow);
            }
            // Сдвигаем текст вниз: вставляем rowsNeeded-1 строк перед textStartRow
            sheet.shiftRows(textStartRow, sheet.getLastRowNum(), rowsNeeded - 1);
            // Копируем стили образца на вставленные строки
            for (int i = 1; i < rowsNeeded; i++) {
                Row newRow = sheet.createRow(firstDataRow + i);
                copyRowStyle(templateRow, newRow);
            }
        }

        // 4. Заполняем данные
        int rowNum = firstDataRow;
        int seq = 1;
        for (RowData data : rows) {
            Row row = sheet.getRow(rowNum);
            if (row == null) row = sheet.createRow(rowNum);

            if (isRvp) {
                fillRvpRow(row, seq++, data);
            } else {
                fillPotRow(row, seq++, data);
            }
            rowNum++;
        }

        // Если данных нет, очищаем образцовую строку
        if (rows.isEmpty()) {
            Row emptyRow = sheet.getRow(firstDataRow);
            if (emptyRow != null) {
                for (Cell cell : emptyRow) {
                    cell.setCellValue("");
                }
            }
        }

        // Перемещаем итоговые формулы, если есть (для ПОТ там итоги)
        if (!isRvp) {
            updatePotTotalRow(sheet, firstDataRow + rows.size());
        }
    }

    /**
     * Поиск строки, где начинается текст под таблицей (ищем "Итого" или "Примечание" в первом столбце).
     */
    private int findTextStartRow(Sheet sheet) {
        DataFormatter formatter = new DataFormatter();
        for (int i = 8; i <= sheet.getLastRowNum(); i++) {
            Row row = sheet.getRow(i);
            if (row == null) continue;
            Cell cell = row.getCell(0);
            String val = formatter.formatCellValue(cell).trim().toLowerCase();
            if (val.startsWith("итого") || val.startsWith("примечание")) {
                return i;
            }
        }
        return -1;
    }

    /**
     * Копирование стилей ячеек из одной строки в другую.
     */
    private void copyRowStyle(Row source, Row target) {
        for (int i = 0; i < source.getLastCellNum(); i++) {
            Cell srcCell = source.getCell(i);
            Cell tgtCell = target.createCell(i);
            if (srcCell != null) {
                tgtCell.setCellStyle(srcCell.getCellStyle());
            }
        }
    }

    /**
     * Заполнение строки листа "Распоряжение (РВП)".
     */
    private void fillRvpRow(Row row, int seq, RowData data) {
        int col = 0;
        // № п/п
        row.createCell(col++).setCellValue(seq);
        // Номер лицевого счета
        row.createCell(col++).setCellValue(data.accountRB);
        // Сумма элемента расчетной базы = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Категория качества = 5
        row.createCell(col++).setCellValue(5);
        // Размер расчетного резерва (%) = 100
        row.createCell(col++).setCellValue(100);
        // Размер расчетного резерва (руб) = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Минимальный размер резерва = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Формируемый резерв = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Доля формируемого резерва = 1
        row.createCell(col++).setCellValue(1);
        // Фактически сформированный резерв = Остаток счета резерва
        setNumericCell(row.createCell(col++), data.reserveBalance);
        // Изменение резерва = Расхождение
        setNumericCell(row.createCell(col++), data.discrepancy);
    }

    /**
     * Заполнение строки листа "Расчетная база ПОТ".
     */
    private void fillPotRow(Row row, int seq, RowData data) {
        int col = 0;
        // № п/п (не указано в ТЗ, но для порядка)
        row.createCell(col++).setCellValue(seq);
        // ФИО клиента

        BigDecimal reserveCoef;
        int qualityCategory;

        if (ACCOUNT_60323.matcher(data.accountRB).matches()) {
            reserveCoef = new BigDecimal("100");
            qualityCategory = 5;

        // Категория качества

            erbBalanceValue = data.balanceRB;
        } else {
            erbBalanceValue = data.overdueBalance;
        }
        setNumericCell(row.createCell(col++), erbBalanceValue);

        // Остаток по счету ЭРБ до 30 дней (резервированию не подлежит)
        // по ТЗ равен столбцу G (balanceRB)
        setNumericCell(row.createCell(col++), data.balanceRB);
    }

    private void setNumericCell(Cell cell, BigDecimal value) {
        if (value == null) value = BigDecimal.ZERO;

        // Применяем числовой формат с двумя знаками (стиль лучше задавать через шаблон)

java
import org.apache.poi.ss.usermodel.*;
import org.apache.poi.ss.util.CellRangeAddress;
import org.apache.poi.xssf.usermodel.XSSFWorkbook;

import java.io.*;
import java.math.BigDecimal;
import java.math.RoundingMode;
import java.util.*;
import java.util.regex.Pattern;

public class CzoProcessor {

    // Индексы столбцов исходного файла (0-based)
    private static final int COL_FILIAL = 1;             // B - Филиал (первые 3 цифры)
    private static final int COL_CLIENT_NAME = 2;        // C - Клиент (ФИО)
    private static final int COL_ACCOUNT_RB = 4;         // E - Счет РБ (лицевой счет)
    private static final int COL_CURRENCY = 5;           // F - Валюта счета
    private static final int COL_BALANCE_RB = 6;         // G - Остаток счета РБ (7)
    private static final int COL_OVERDUE_BALANCE = 7;    // H - Остаток просроченной РБ (8)
    private static final int COL_RESERVE_BALANCE = 14;   // O - Остаток счета резерва (15)
    private static final int COL_CALC_RESERVE = 15;      // P - Расчетный резерв (16)
    private static final int COL_DISCREPANCY = 16;       // Q - Расхождение (17)

    // Шаблоны счетов
    private static final Pattern ACCOUNT_60323 = Pattern.compile("^60323.*");
    private static final Pattern ACCOUNT_458 = Pattern.compile("^458.*");
    private static final Pattern ACCOUNT_47423 = Pattern.compile("^47423.*");

    /**
     * Главный метод обработки.
     */
    public void process(File inputFile) throws IOException {
        // 1. Чтение и группировка исходных данных
        Map<String, List<RowData>> groupedByFilial = readAndGroupByFilial(inputFile);

        // 2. Для каждого филиала создаём файл на основе шаблона
        for (Map.Entry<String, List<RowData>> entry : groupedByFilial.entrySet()) {
            String filialCode = entry.getKey();
            List<RowData> rows = entry.getValue();
            createOrderFromTemplate(inputFile.getParentFile(), filialCode, rows);
        }
    }

    /**
     * Чтение исходного Excel, группировка по первым трём цифрам филиала.
     */
    private Map<String, List<RowData>> readAndGroupByFilial(File file) throws IOException {
        Map<String, List<RowData>> groups = new HashMap<>();

        try (FileInputStream fis = new FileInputStream(file);
             Workbook workbook = WorkbookFactory.create(fis)) {

            Sheet sheet = workbook.getSheetAt(0);
            DataFormatter formatter = new DataFormatter();

            for (int rowIdx = 1; rowIdx <= sheet.getLastRowNum(); rowIdx++) {
                Row row = sheet.getRow(rowIdx);
                if (row == null) continue;

                // Филиал (первые 3 символа столбца B)
                Cell filialCell = row.getCell(COL_FILIAL);
                String filialRaw = formatter.formatCellValue(filialCell).trim();
                if (filialRaw.isEmpty()) continue;

                String filialKey = filialRaw.length() >= 3 ? filialRaw.substring(0, 3) : filialRaw;
                if (!isTargetFilial(filialKey)) continue;

                RowData data = new RowData();
                data.accountRB = formatter.formatCellValue(row.getCell(COL_ACCOUNT_RB));
                data.clientName = formatter.formatCellValue(row.getCell(COL_CLIENT_NAME));
                data.currency = formatter.formatCellValue(row.getCell(COL_CURRENCY));
                data.balanceRB = getNumericValue(row.getCell(COL_BALANCE_RB));
                data.overdueBalance = getNumericValue(row.getCell(COL_OVERDUE_BALANCE));
                data.reserveBalance = getNumericValue(row.getCell(COL_RESERVE_BALANCE));
                data.calcReserve = getNumericValue(row.getCell(COL_CALC_RESERVE));
                data.discrepancy = getNumericValue(row.getCell(COL_DISCREPANCY));

                groups.computeIfAbsent(filialKey, k -> new ArrayList<>()).add(data);
            }
        }
        return groups;
    }

    private boolean isTargetFilial(String code) {
        try {
            int num = Integer.parseInt(code);
            return (num >= 1 && num <= 3) || (num >= 5 && num <= 9);
        } catch (NumberFormatException e) {
            return false;
        }
    }

    private BigDecimal getNumericValue(Cell cell) {
        if (cell == null) return BigDecimal.ZERO;
        try {
            if (cell.getCellType() == CellType.NUMERIC) {
                return BigDecimal.valueOf(cell.getNumericCellValue());
            } else if (cell.getCellType() == CellType.STRING) {
                String str = cell.getStringCellValue().trim().replace(",", ".");
                return new BigDecimal(str.isEmpty() ? "0" : str);
            }
        } catch (Exception ignored) {}
        return BigDecimal.ZERO;
    }

    /**
     * Создание файла распоряжения из шаблона template.xlsx.
     */
    private void createOrderFromTemplate(File parentDir, String filialCode, List<RowData> allRows) throws IOException {
        // Загружаем шаблон из ресурсов
        try (InputStream templateStream = getClass().getResourceAsStream("/template.xlsx")) {
            if (templateStream == null) {
                throw new IOException("Шаблон template.xlsx не найден в ресурсах!");
            }

            try (Workbook workbook = new XSSFWorkbook(templateStream)) {
                // Фильтрация строк для двух листов
                List<RowData> rvpRows = new ArrayList<>();
                List<RowData> potRows = new ArrayList<>();

                for (RowData row : allRows) {
                    // РВП: счета 60323*,458*,47423* и расхождение != 0
                    if (isRvpAccount(row.accountRB) && row.discrepancy.compareTo(BigDecimal.ZERO) != 0) {
                        rvpRows.add(row);
                    }

                    // ПОТ
                    if (isPotAccount(row.accountRB)) {
                        if (ACCOUNT_60323.matcher(row.accountRB).matches()) {
                            if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0) {
                                potRows.add(row);
                            }
                        } else { // 458* или 47423*
                            if (row.balanceRB.compareTo(BigDecimal.ZERO) > 0 ||
                                row.overdueBalance.compareTo(BigDecimal.ZERO) > 0) {
                                potRows.add(row);
                            }
                        }
                    }
                }

                // Заполняем листы
                Sheet sheetRvp = workbook.getSheet("Распоряжение (РВП)");
                Sheet sheetPot = workbook.getSheet("Расчетная база ПОТ");
                if (sheetRvp == null || sheetPot == null) {
                    throw new IOException("В шаблоне отсутствуют нужные листы!");
                }

                fillSheetWithInsert(sheetRvp, rvpRows, true);
                fillSheetWithInsert(sheetPot, potRows, false);

                // Сохраняем результат
                String outputFileName = String.format("Распоряжение_филиал_%s.xlsx", filialCode);
                File outputFile = new File(parentDir, outputFileName);
                try (FileOutputStream fos = new FileOutputStream(outputFile)) {
                    workbook.write(fos);
                }
            }
        }
    }

    private boolean isRvpAccount(String account) {
        return ACCOUNT_60323.matcher(account).matches() ||
               ACCOUNT_458.matcher(account).matches() ||
               ACCOUNT_47423.matcher(account).matches();
    }

    private boolean isPotAccount(String account) {
        return isRvpAccount(account);
    }

    /**
     * Заполнение листа с динамической вставкой строк.
     * @param sheet лист
     * @param rows данные
     * @param isRvp true для листа "Распоряжение (РВП)", false для "Расчетная база ПОТ"
     */
    private void fillSheetWithInsert(Sheet sheet, List<RowData> rows, boolean isRvp) {
        // 1. Найти строку, с которой начинается текст под таблицей (после 8-й)
        int textStartRow = findTextStartRow(sheet);
        if (textStartRow == -1) {
            // Если не нашли, считаем, что текст начинается сразу после шаблонной строки (10)
            textStartRow = 9; // индекс 9 = 10-я строка
        }

        // 2. Удаляем все строки между 9-й (индекс 8) и textStartRow (не включая textStartRow)
        int firstDataRow = 8; // 9-я строка (индекс 8)
        if (textStartRow > firstDataRow) {
            for (int i = textStartRow - 1; i >= firstDataRow; i--) {
                Row row = sheet.getRow(i);
                if (row != null) {
                    sheet.removeRow(row);
                }
            }
        }

        // 3. Вставляем столько строк, сколько нужно для данных (если rows.size() > 1)
        int rowsNeeded = Math.max(rows.size(), 1);
        if (rowsNeeded > 1) {
            // Копируем стили из строки-образца (сейчас она на месте 9-й строки)
            Row templateRow = sheet.getRow(firstDataRow);
            if (templateRow == null) {
                templateRow = sheet.createRow(firstDataRow);
            }
            // Сдвигаем текст вниз: вставляем rowsNeeded-1 строк перед textStartRow
            sheet.shiftRows(textStartRow, sheet.getLastRowNum(), rowsNeeded - 1);
            // Копируем стили образца на вставленные строки
            for (int i = 1; i < rowsNeeded; i++) {
                Row newRow = sheet.createRow(firstDataRow + i);
                copyRowStyle(templateRow, newRow);
            }
        }

        // 4. Заполняем данные
        int rowNum = firstDataRow;
        int seq = 1;
        for (RowData data : rows) {
            Row row = sheet.getRow(rowNum);
            if (row == null) row = sheet.createRow(rowNum);

            if (isRvp) {
                fillRvpRow(row, seq++, data);
            } else {
                fillPotRow(row, seq++, data);
            }
            rowNum++;
        }

        // Если данных нет, очищаем образцовую строку
        if (rows.isEmpty()) {
            Row emptyRow = sheet.getRow(firstDataRow);
            if (emptyRow != null) {
                for (Cell cell : emptyRow) {
                    cell.setCellValue("");
                }
            }
        }

        // Перемещаем итоговые формулы, если есть (для ПОТ там итоги)
        if (!isRvp) {
            updatePotTotalRow(sheet, firstDataRow + rows.size());
        }
    }

    /**
     * Поиск строки, где начинается текст под таблицей (ищем "Итого" или "Примечание" в первом столбце).
     */
    private int findTextStartRow(Sheet sheet) {
        DataFormatter formatter = new DataFormatter();
        for (int i = 8; i <= sheet.getLastRowNum(); i++) {
            Row row = sheet.getRow(i);
            if (row == null) continue;
            Cell cell = row.getCell(0);
            String val = formatter.formatCellValue(cell).trim().toLowerCase();
            if (val.startsWith("итого") || val.startsWith("примечание")) {
                return i;
            }
        }
        return -1;
    }

    /**
     * Копирование стилей ячеек из одной строки в другую.
     */
    private void copyRowStyle(Row source, Row target) {
        for (int i = 0; i < source.getLastCellNum(); i++) {
            Cell srcCell = source.getCell(i);
            Cell tgtCell = target.createCell(i);
            if (srcCell != null) {
                tgtCell.setCellStyle(srcCell.getCellStyle());
            }
        }
    }

    /**
     * Заполнение строки листа "Распоряжение (РВП)".
     */
    private void fillRvpRow(Row row, int seq, RowData data) {
        int col = 0;
        // № п/п
        row.createCell(col++).setCellValue(seq);
        // Номер лицевого счета
        row.createCell(col++).setCellValue(data.accountRB);
        // Сумма элемента расчетной базы = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Категория качества = 5
        row.createCell(col++).setCellValue(5);
        // Размер расчетного резерва (%) = 100
        row.createCell(col++).setCellValue(100);
        // Размер расчетного резерва (руб) = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Минимальный размер резерва = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Формируемый резерв = Расчетный резерв
        setNumericCell(row.createCell(col++), data.calcReserve);
        // Доля формируемого резерва = 1
        row.createCell(col++).setCellValue(1);
        // Фактически сформированный резерв = Остаток счета резерва
        setNumericCell(row.createCell(col++), data.reserveBalance);
        // Изменение резерва = Расхождение
        setNumericCell(row.createCell(col++), data.discrepancy);
    }

    /**
     * Заполнение строки листа "Расчетная база ПОТ".
     */
    private void fillPotRow(Row row, int seq, RowData data) {
        int col = 0;
        // № п/п (не указано в ТЗ, но для порядка)
        row.createCell(col++).setCellValue(seq);
        // ФИО клиента
        row.createCell(col++).setCellValue(data.clientName);
        // Номер договора (нет в исходных данных — оставляем пустым)
        row.createCell(col++).setCellValue("");
        // Дата заключения договора (нет данных)
        row.createCell(col++).setCellValue("");
        // Признак принадлежности к ПОТ на предыдущую дату (нет)
        row.createCell(col++).setCellValue("");
        // Признак принадлежности к ПОТ на дату отчета (нет)
        row.createCell(col++).setCellValue("");
        // Валюта
        row.createCell(col++).setCellValue(data.currency);

        // === Коэффициент резервирования и Категория качества ===
        BigDecimal reserveCoef;
        int qualityCategory;

        if (ACCOUNT_60323.matcher(data.accountRB).matches()) {
            reserveCoef = new BigDecimal("100");
            qualityCategory = 5;
        } else { // 458* или 47423*
            // Остаток по счету ЭРБ – в зависимости от счёта
            BigDecimal erbBalance = ACCOUNT_458.matcher(data.accountRB).matches() ||
                                    ACCOUNT_47423.matcher(data.accountRB).matches()
                                    ? data.overdueBalance   // для 458/47423 берём просроченный остаток
                                    : data.balanceRB;       // для остальных (не должно быть) – основной остаток
            if (erbBalance.compareTo(BigDecimal.ZERO) > 0) {
                reserveCoef = new BigDecimal("100");
                qualityCategory = 5;
            } else {
                reserveCoef = BigDecimal.ZERO;
                qualityCategory = 1;
            }
        }
        // Коэффициент резервирования
        setNumericCell(row.createCell(col++), reserveCoef);
        // Категория качества
        row.createCell(col++).setCellValue(qualityCategory);

        // Счет для учета ЭРБ = Счет РБ
        row.createCell(col++).setCellValue(data.accountRB);

        // Остаток по счету ЭРБ
        BigDecimal erbBalanceValue;
        if (ACCOUNT_60323.matcher(data.accountRB).matches()) {
            erbBalanceValue = data.balanceRB;
        } else {
            erbBalanceValue = data.overdueBalance;
        }
        setNumericCell(row.createCell(col++), erbBalanceValue);

        // Остаток по счету ЭРБ до 30 дней (резервированию не подлежит)
        // по ТЗ равен столбцу G (balanceRB)
        setNumericCell(row.createCell(col++), data.balanceRB);
    }

        if (value == null) value = BigDecimal.ZERO;
        cell.setCellValue(value.doubleValue());

    }

Пояснения
Шаблон должен находиться в src/main/resources/template.xlsx.

Поиск текста под таблицей: метод findTextStartRow ищет в первом столбце слова "итого" или "примечание" (регистронезависимо) и возвращает индекс этой строки. Перед ней будут вставлены данные.

Вставка строк: если записей больше одной, метод sheet.shiftRows сдвигает весь нижний текст вниз, а затем создаются новые строки с копированием стилей из строки-образца (9-й строки).

Формулы расчёта коэффициента резервирования и категории качества реализованы в точности по предоставленному ТЗ.

Числовые значения записываются как double, стиль ячеек (число десятичных знаков) должен быть настроен в шаблоне.

Использование
Поместите template.xlsx в папку resources.

Соберите проект (например, через Maven) и запустите.

Нажмите кнопку, выберите исходный файл CZO.

В папке с исходным файлом появятся файлы вида Распоряжение_филиал_001.xlsx и т.д.

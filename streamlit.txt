Что такое защитный код?

Это код, который НЕ ДАЕТ ПРИЛОЖЕНИЮ УПАСТЬ, даже если в базе данных нет нужной колонки.

Проблема без защиты:

```python
def get_vsp_by_filial(self, filial_id):
    # Если колонки close_vsp нет - ПРИЛОЖЕНИЕ УПАДЕТ с ошибкой!
    return self._to_df(f"""
        SELECT id, name, name_vsp 
        FROM vsp 
        WHERE filial_id=%s AND close_vsp = false 
        ORDER BY name
    """, (filial_id,))
```

Ошибка: ERROR: column "close_vsp" does not exist

---

Решение с защитным кодом:

Вариант 1: try/except (ПРОСТОЙ)

```python
def get_vsp_by_filial(self, filial_id):
    try:
        # Пытаемся выполнить запрос с close_vsp
        return self._to_df(f"""
            SELECT id, name, name_vsp 
            FROM {self._table_name('vsp')} 
            WHERE filial_id=%s AND close_vsp = false 
            ORDER BY name
        """, (int(filial_id),))
    except Exception as e:
        # Если колонки нет - делаем запрос БЕЗ close_vsp
        print(f"⚠️ Колонка close_vsp не найдена, работаем без фильтра. Ошибка: {e}")
        return self._to_df(f"""
            SELECT id, name, name_vsp 
            FROM {self._table_name('vsp')} 
            WHERE filial_id=%s 
            ORDER BY name
        """, (int(filial_id),))
```

Как работает:

1. Пытается выполнить запрос с close_vsp
2. Если колонка есть → работает с фильтром
3. Если колонки нет → ловит ошибку и выполняет запрос без фильтра
4. Приложение НЕ ПАДАЕТ!

Вариант 2: Проверка при запуске (БОЛЕЕ ЭФФЕКТИВНЫЙ)

Добавьте в класс DatabaseManager в __init__:

```python
def __init__(self):
    self.schema = PG_CONFIG['schema']
    self._connection = None
    self._cursor = None
    
    # ПРОВЕРЯЕМ НАЛИЧИЕ КОЛОНКИ ПРИ ЗАПУСКЕ
    self.has_close_vsp = self._check_column_exists('vsp', 'close_vsp')
    
def _check_column_exists(self, table, column):
    """Проверяет, существует ли колонка в таблице"""
    try:
        result = self._execute(f"""
            SELECT EXISTS (
                SELECT 1 
                FROM information_schema.columns 
                WHERE table_schema = %s 
                AND table_name = %s 
                AND column_name = %s
            )
        """, (self.schema, table, column), fetch_one=True)
        return result['exists'] if result else False
    except:
        return False
```

И используем:

